    - 1️⃣ What is the difference between var, let, and const?
    ans : const is for fixed values; let is for values that change; var is the outdated version you should generally avoid.
    - 2️⃣ What is the spread operator (...)?
     ans : The "unpacker." It takes all items out of an array or object so you can copy or combine them easily.
    - 3️⃣ What is the difference between map(), filter(), and forEach()?
    ans : map creates a transformed new array; filter creates a new array with only specific items; forEach just loops without returning anything.
    - 4️⃣ What is an arrow function?
   ans : A shorter, cleaner way to write functions using => instead of the function keyword.
    - 5️⃣ What are template literals?
   ans : Strings wrapped in backticks that allow you to plug in variables using and write on multiple lines.
